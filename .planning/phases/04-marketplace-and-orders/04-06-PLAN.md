  <action>
    **apps/web/src/contexts/cart-context.tsx (CartProvider):**
    - Create a React context `CartContext` with `CartProvider` and `useCart` hook
    - State: `cartCount` (number), `cartItems` (array), `isLoading` (boolean)
    - Methods: `refreshCart()` — calls getCart() and updates cartCount + cartItems; `incrementCount()` — optimistically increments cartCount by 1 (called after successful addToCart from any component); `decrementCount()` — optimistically decrements cartCount by 1 (called after removeFromCart)
    - On mount (when user is authenticated): call getCartSummary() to initialize cartCount
    - The provider wraps the app layout so all components (header badge, cart sidebar, listing detail "Add to Cart" button) share the same cart state
    - This ensures the header cart badge updates immediately when items are added from the listing detail page or removed from the cart sidebar, without requiring a page refresh

    **cart-countdown.tsx:**
    - Custom hook `useCartCountdown(expiresAt: string)` that returns { hours, minutes, seconds, isExpired, remaining }
    - Uses setInterval(1000ms) with cleanup
    - Renders as "XXh XXm XXs" in normal state, red "Expirado" when expired
    - Yellow warning color when < 1 hour remaining

    **cart-item-card.tsx:**
    - Shows: cover image thumbnail, title, condition badge, seller name, price (R$ formatted)
    - CartCountdown component showing remaining reservation time
    - Remove button (X icon) — calls removeFromCart, shows toast on success
    - If expired (isExpired from countdown): grayed out styling, "Reserva expirada" label, no checkout allowed
    - Links to listing detail page on click

    **cart-summary.tsx:**
    - Shows total items count, total amount in BRL
    - "Limpar Carrinho" button to clear all items
    - "Finalizar Compra" button linking to /checkout
    - Disabled if any items are expired or cart is empty

    **cart-sidebar.tsx:**
    - Sheet component (shadcn) sliding from right side
    - Triggered by cart icon in header
    - Uses `useCart().refreshCart()` on Sheet open (onOpenChange) to sync full cart data
    - After removeFromCart: calls `useCart().decrementCount()` for instant badge update
    - After clearCart: calls `useCart().refreshCart()` to reset count to 0
    - Renders list of CartItemCard components
    - CartSummary at bottom
    - Empty state: "Seu carrinho esta vazio" with link to marketplace

    **header.tsx modification:**
    - Add cart icon (ShoppingCart from lucide) in the header bar next to user menu
    - Show badge with item count from `useCart().cartCount` (from CartProvider — NOT a direct API call)
    - Click opens cart-sidebar Sheet
    - Cart icon only visible when user is authenticated
    - Badge auto-updates when CartProvider state changes (e.g., after addToCart from listing detail or removeFromCart from sidebar)

    **i18n additions to pt-BR.json:**
    Add `cart` section: title: "Carrinho", empty: "Seu carrinho esta vazio", browseMarketplace: "Explorar marketplace", itemCount: "{count} {count, plural, one {item} other {itens}}", total: "Total", clearCart: "Limpar Carrinho", checkout: "Finalizar Compra", reservationExpires: "Reserva expira em", reservationExpired: "Reserva expirada", removeItem: "Remover item", itemRemoved: "Item removido do carrinho", cartCleared: "Carrinho limpo"
