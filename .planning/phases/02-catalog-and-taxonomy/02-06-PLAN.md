**Catalog API Service (`apps/web/src/lib/api/catalog.ts`):**
- Import apiClient from client.ts
- `searchCatalog(params: CatalogSearchParams)`: GET `/api/v1/catalog` with query params (title, publisher, seriesId, categoryIds as comma-joined, characterIds as comma-joined, tagIds as comma-joined, yearFrom, yearTo, sortBy, sortOrder, page, limit). Return `{ data: CatalogEntry[], pagination }`.
- `getCatalogEntryById(id: string)`: GET `/api/v1/catalog/${id}`. Return full CatalogEntry.
- Define local TypeScript interfaces: CatalogEntry (id, title, author, publisher, imprint, barcode, isbn, description, coverImageUrl, approvalStatus, averageRating, ratingCount, volumeNumber, editionNumber, series, categories, tags, characters, createdAt), with nested Category/Tag/Character types.

**Taxonomy API Service (`apps/web/src/lib/api/taxonomy.ts`):**
- `getCategories()`: GET `/api/v1/categories`. Return `Category[]`.
- `getTags()`: GET `/api/v1/tags`. Return `Tag[]`.
- `getCharacters(page?: number, limit?: number)`: GET `/api/v1/characters`. Return `{ data: Character[], pagination }`.
- `getSeries(params?)`: If series API service from Plan 05 exists, this can re-export from there. Otherwise create GET `/api/v1/series` wrapper here.

**Star Rating Component (`apps/web/src/components/features/catalog/star-rating.tsx`):**
- Props: `rating: number` (0-5, can be decimal), `count: number`, `size?: 'sm' | 'md' | 'lg'`
- Display: 5 star icons (Star from lucide-react). Fill proportionally: full stars (rounded down), half star if decimal >= 0.5 (use half-filled approach or just round), empty stars for remainder.
- Show rating number (e.g., "4.2") and count in parentheses ("(15 avaliacoes)") next to stars
- If count is 0, show empty stars and "Sem avaliacoes" text
- Use yellow-500 for filled stars, muted color for empty

**Catalog Card Component (`apps/web/src/components/features/catalog/catalog-card.tsx`):**
- Props: `entry: CatalogEntry`
- Display: cover image (aspect-[2/3] container, gray placeholder with BookOpen icon if no coverImageUrl), title (h3, truncated to 2 lines), author, publisher, series badge (if has series), StarRating component, categories as small badges (first 3, "+N more" if more)
- Link to `/catalog/${entry.id}` using next/link
- Use Card from shadcn/ui with hover shadow transition
- Responsive sizing

**Catalog Filters Component (`apps/web/src/components/features/catalog/catalog-filters.tsx`):**
- Props: `filters: CatalogSearchParams`, `onFiltersChange: (filters) => void`, `categories: Category[]`, `characters: Character[]`, `series: SeriesOption[]`
- Sections:
  - Title search: Input with debounce (300ms)
  - Publisher search: Input with debounce (300ms)
  - Series: Select dropdown (shadcn Select) with "Todas as series" default
  - Categories: Checkbox list (shadcn Checkbox) â€” show all, user can check multiple
  - Characters: Checkbox list with search input to filter long list
  - Year range: Two number inputs (de/ate)
  - Sort by: Select dropdown (Titulo, Data, Avaliacao)
  - Sort order: Toggle button (Crescente/Decrescente)
  - Clear filters button
- On any change, call onFiltersChange with updated params
- Collapsible sections on mobile (use Collapsible from shadcn/ui)
