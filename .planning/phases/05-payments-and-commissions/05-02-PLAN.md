  <action>
Create `apps/api/src/modules/banking/banking.service.ts`:

Import `cpf` from `cpf-cnpj-validator`. Import contracts types from `@comicstrunk/contracts`.

1. `createBankAccount(userId: string, data: CreateBankAccountInput)`:
   - Validate CPF: `if (!cpf.isValid(data.cpf)) throw new BadRequestError('Invalid CPF')`.
   - Strip CPF formatting: `const cleanCpf = cpf.strip(data.cpf)`.
   - If `data.isPrimary` is true: use `$transaction` to unset all existing isPrimary for this userId, then create.
   - If `data.isPrimary` is false and this is the user's first account: set isPrimary to true automatically.
   - Create BankAccount record. Return it.

2. `updateBankAccount(userId: string, accountId: string, data: UpdateBankAccountInput)`:
   - Find account by id. Validate ownership (userId match).
   - If CPF provided in update: validate with `cpf.isValid()` and strip formatting.
   - If `isPrimary` changing to true: use `$transaction` to unset all other primary accounts first.
   - If `isPrimary` changing to false: prevent if it's the only account (throw BadRequestError 'Cannot unset primary on only account').
   - Update and return.

3. `deleteBankAccount(userId: string, accountId: string)`:
   - Find account by id. Validate ownership.
   - Delete the account.
   - If deleted account was primary and user still has other accounts: auto-promote the most recent (createdAt desc) to primary. Pattern matches Phase 4 shipping address default promotion.

4. `listUserBankAccounts(userId: string)`:
   - `prisma.bankAccount.findMany({ where: { userId }, orderBy: { createdAt: 'desc' } })`.

5. `getBankAccount(userId: string, accountId: string)`:
   - Find by id, validate ownership, return.

6. `setPrimaryBankAccount(userId: string, accountId: string)`:
   - Find account, validate ownership.
   - Use `$transaction`: unset all isPrimary for userId, set this one as primary.

7. Admin functions:
   - `adminListBankAccounts(filters: { userId?: string, page: number, limit: number })`:
     - List all bank accounts with user info (select: name, email).
     - Filter by userId if provided. Paginate.
     - Return `{ accounts, total, page, limit }`.
