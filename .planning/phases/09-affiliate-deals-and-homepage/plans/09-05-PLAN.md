# Plan 09-05: Homepage UI

## Wave: 2 (depends on 09-01 + 09-03)

## Goal
Replace the current homepage redirect with a real configurable homepage showing banners, catalog highlights, deals of the day, and featured coupons.

## Tasks

### 1. Create homepage API client
In `apps/web/src/lib/api/homepage.ts`:
- getHomepageData() — GET /api/v1/homepage
- Admin functions (for 09-06)

### 2. Create homepage section components
In `apps/web/src/components/features/homepage/`:

- `homepage-banner-carousel.tsx` — Full-width carousel with deal/catalog banners. Auto-rotate every 5s. Dots indicator. Link to deal or catalog entry. Use shadcn Carousel or simple CSS transitions.

- `homepage-catalog-highlights.tsx` — Grid of 4-8 catalog entry cards showing: cover image, title, series name, average rating stars, "Ver detalhes" link. Horizontal scrollable on mobile.

- `homepage-deals-of-day.tsx` — Horizontal strip of 3-6 deal cards with countdown to expiry. "Ver todas as ofertas →" link to /deals. Affiliate disclosure inline.

- `homepage-featured-coupons.tsx` — Row of coupon cards with: store logo, coupon code (copy button), discount text, "Usar cupom" link. Max 4-6 coupons.

- `homepage-section-renderer.tsx` — Takes a section object (type + data) and renders the correct component. Handles unknown types gracefully.

### 3. Replace homepage page
Update `apps/web/src/app/[locale]/(public)/page.tsx`:
- Remove the redirect to /catalog
- Fetch homepage data from API
- Render each section via homepage-section-renderer
- Fallback: if no sections configured, show catalog highlights as default
- Hero section at top with "Comics Trunk" branding + tagline

### 4. Add SEO metadata
- Page title: "Comics Trunk — Plataforma para Colecionadores de Quadrinhos"
- Meta description in PT-BR

## Verification
- Homepage shows configurable sections
- Carousel auto-rotates
- Catalog highlights link to catalog detail
- Deals link to click tracking URL
- Coupons have copy-to-clipboard
- Responsive on mobile
