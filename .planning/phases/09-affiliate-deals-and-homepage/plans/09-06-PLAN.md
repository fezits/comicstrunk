# Plan 09-06: Admin Deals + Homepage UI

## Wave: 2 (parallel with 09-04 + 09-05, depends on Wave 1)

## Goal
Create admin pages for managing partner stores, deals, click analytics, and homepage section configuration.

## Tasks

### 1. Extend deals API client for admin
In `apps/web/src/lib/api/deals.ts` (extend existing):
- Admin partner store: listAll, create, update, delete
- Admin deals: listAll, create, update, delete, uploadBanner
- Admin analytics: getAnalytics, exportCSV
- Admin homepage: listSections, createSection, updateSection, deleteSection, reorderSections

### 2. Create admin deal components
In `apps/web/src/components/features/deals/`:

- `admin-partner-stores.tsx` — Table of partner stores with: name, slug, affiliateTag, baseUrl, active status. Create/edit dialog with form fields. Toggle active status. Delete confirmation.

- `admin-deals-list.tsx` — Table of all deals: title, store, type, discount, status (ativa/expirada/inativa), dates. Filter by store/type/status. Create button. Each row: edit, delete, toggle active.

- `admin-deal-form.tsx` — Create/edit deal form (Dialog or page):
  - Store select (from partner stores)
  - Type radio (Cupom/Promoção)
  - Title, description, discount text
  - Coupon code (if COUPON type)
  - Affiliate base URL
  - Category select (optional)
  - Start date, expiry date (date pickers)
  - Banner image upload
  - Preview of composed affiliate URL

- `admin-click-analytics.tsx` — Analytics dashboard:
  - Date range picker (último 7 dias, 30 dias, personalizado)
  - Cards: Total de cliques, Cliques únicos, Ofertas ativas
  - Table: Oferta, Loja, Categoria, Cliques, Únicos
  - "Exportar CSV" button
  - Simple bar chart (CSS-based, no chart library needed) showing clicks by store

### 3. Create admin homepage config components
In `apps/web/src/components/features/homepage/`:

- `admin-homepage-sections.tsx` — Drag-and-drop (or up/down arrows) list of homepage sections. Each section shows: type icon, title, visibility toggle, sort order. Add new section button. Delete section. Click to edit content refs.

- `admin-section-editor.tsx` — Edit a section's content:
  - For BANNER_CAROUSEL: select deals or catalog entries for banners
  - For CATALOG_HIGHLIGHTS: select catalog entries
  - For DEALS_OF_DAY: auto (expiring soonest) or manual deal selection
  - For FEATURED_COUPONS: auto (active coupons) or manual selection
  - Title override field
  - Visibility toggle

### 4. Create admin route pages
- `apps/web/src/app/[locale]/(admin)/admin/deals/page.tsx` — Partner stores + deals list + analytics tabs
- `apps/web/src/app/[locale]/(admin)/admin/homepage/page.tsx` — Homepage section management

### 5. Add admin nav links
- Add "Ofertas" and "Homepage" to admin nav in nav-config.ts
- Add PT-BR translations

## Verification
- Admin can manage partner stores (CRUD)
- Admin can manage deals (CRUD with banner upload)
- Analytics show click data with export
- Admin can configure homepage sections (order, visibility, content)
