  <action>
1. **Create `apps/api/src/modules/subscriptions/subscriptions.service.ts`**:

   **`listActivePlans()`**: Query PlanConfig where isActive=true, ordered by planType asc, price asc. Return all fields.

   **`getSubscriptionStatus(userId: string)`**: Find most recent subscription for user where status IN ('ACTIVE', 'TRIALING'). If none found, return a synthetic FREE status object with planType='FREE', status='ACTIVE', no stripe IDs, no period dates. Include plan config details (collectionLimit, commissionRate) by querying PlanConfig for the user's planType.

   **`createCheckoutSession(userId: string, planConfigId: string)`**:
   - Look up PlanConfig by id, verify isActive=true
   - If no stripePriceId on the config, throw BadRequestError('Plan not configured for Stripe billing')
   - Check if user already has ACTIVE or TRIALING subscription with planType != 'FREE'. If so, throw BadRequestError('You already have an active subscription. Use the customer portal to manage it.')
   - Check for existing stripeCustomerId in any of user's Subscription records
   - If no stripeCustomerId, create Stripe customer via `stripe.customers.create({ email, name, metadata: { comicstrunkUserId } })`
   - Create Stripe Checkout Session with:
     - customer: stripeCustomerId
     - line_items: [{ price: planConfig.stripePriceId, quantity: 1 }]
     - mode: 'subscription'
     - success_url: `${WEB_URL}/subscription/success?session_id={CHECKOUT_SESSION_ID}`
     - cancel_url: `${WEB_URL}/subscription/cancel`
     - subscription_data.trial_period_days: planConfig.trialDays || undefined
     - subscription_data.metadata: { comicstrunkUserId: userId }
     - metadata: { comicstrunkUserId: userId, planConfigId }
   - If stripe not configured (isStripeConfigured()===false), throw BadRequestError('Stripe is not configured')
   - Return { url: session.url, sessionId: session.id }

   **`createPortalSession(userId: string)`**:
   - Find user's subscription with stripeCustomerId not null
   - If none, throw BadRequestError('No Stripe subscription found')
   - Create portal session via `stripe.billingPortal.sessions.create({ customer: stripeCustomerId, return_url: WEB_URL + '/account/subscription' })`
   - Return { url: session.url }

   **`cancelSubscription(userId: string)`**:
   - Find user's ACTIVE or TRIALING subscription with stripeSubscriptionId
   - If none, throw NotFoundError
   - Call `stripe.subscriptions.update(stripeSubId, { cancel_at_period_end: true })`
   - Update local subscription: cancelledAt = new Date()
   - Return updated subscription

2. **Create `apps/api/src/modules/subscriptions/subscriptions.routes.ts`**:

   - POST `/checkout` (authenticate, validate(createCheckoutSchema)) — calls createCheckoutSession, returns via sendSuccess with 201
   - POST `/portal` (authenticate) — calls createPortalSession, returns via sendSuccess
   - POST `/cancel` (authenticate) — calls cancelSubscription, returns via sendSuccess
