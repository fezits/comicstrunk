1. **Create `apps/web/src/components/ui/star-rating.tsx`**:
   - Props: `rating: number` (0-5, supports decimals for display), `maxStars?: number` (default 5), `size?: 'sm' | 'md' | 'lg'`, `interactive?: boolean`, `onChange?: (rating: number) => void`, `className?: string`
   - Display mode (interactive=false): Renders filled, half-filled, and empty stars based on rating value. Uses Star icon from lucide-react with fill for full stars and half-fill technique for partial.
   - Input mode (interactive=true): Stars are clickable/hoverable. On hover, highlights stars up to hover position. On click, calls onChange with selected rating (1-5 integers).
   - Size variants: sm=16px, md=20px, lg=24px
   - Color: filled stars use amber/yellow (#F59E0B), empty stars use muted foreground
   - Accessible: aria-label with rating value, role="radiogroup" for interactive mode

2. **Create `apps/web/src/lib/api/reviews.ts`**:
   API client functions using the existing axios instance from `lib/api-client.ts`:
   - `createCatalogReview(data)` — POST /api/v1/reviews/catalog
   - `createSellerReview(data)` — POST /api/v1/reviews/seller
   - `updateReview(id, data)` — PUT /api/v1/reviews/:id
   - `deleteReview(id)` — DELETE /api/v1/reviews/:id
   - `getCatalogReviews(catalogEntryId, params)` — GET /api/v1/reviews/catalog/:id
   - `getSellerReviews(sellerId, params)` — GET /api/v1/reviews/seller/:id
   - `getUserReviewForCatalog(catalogEntryId)` — GET /api/v1/reviews/catalog/:id/mine

3. **Create `apps/web/src/lib/api/comments.ts`**:
   - `createComment(data)` — POST /api/v1/comments
   - `updateComment(id, data)` — PUT /api/v1/comments/:id
   - `deleteComment(id)` — DELETE /api/v1/comments/:id
   - `getCatalogComments(catalogEntryId, params)` — GET /api/v1/comments/catalog/:id
   - `toggleCommentLike(commentId)` — POST /api/v1/comments/:id/like
  </action>
