<objective>
Build the order management UI for both buyers and sellers: buyer order history and detail pages with status timelines and per-item tracking, seller order dashboard with tracking code entry form.

Purpose: Buyers need visibility into their orders with clear status progression. Sellers need a dedicated view of orders they need to fulfill with a tracking code entry form. Per-item tracking supports multi-seller orders where each item ships independently.
Output: Buyer order pages (list + detail) in (orders) route group, seller order pages (list + detail with tracking) in (seller) route group.
</objective>
