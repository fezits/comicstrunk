- `createCatalogEntrySchema`: title (required, 1-255, trim), author (optional, max 255, trim), publisher (optional, max 255, trim), imprint (optional, max 255, trim), barcode (optional, max 50, trim), isbn (optional, max 20, trim), description (optional, max 5000, trim), seriesId (optional, cuid), volumeNumber (optional, int, positive), editionNumber (optional, int, positive), categoryIds (optional, array of cuid strings), tagIds (optional, array of cuid strings), characterIds (optional, array of cuid strings)

**Install dependencies:**
```bash
pnpm --filter api add multer cloudinary papaparse slugify
pnpm --filter api add -D @types/multer @types/papaparse
```

**cloudinary.ts** (`apps/api/src/shared/lib/cloudinary.ts`):
- Import `v2 as cloudinary` from `cloudinary`
- Configure using `CLOUDINARY_CLOUD_NAME`, `CLOUDINARY_API_KEY`, `CLOUDINARY_API_SECRET` env vars
- If any env var is missing, set a `cloudinaryConfigured` flag to false (do NOT crash — allow dev mode without Cloudinary)
- Export `uploadImage(buffer: Buffer, folder: string): Promise<{ url: string; publicId: string }>` using `cloudinary.uploader.upload_stream` with `quality: 'auto'` and `fetch_format: 'auto'` transformations
- Export `deleteImage(publicId: string): Promise<void>` using `cloudinary.uploader.destroy`
- If `cloudinaryConfigured` is false, `uploadImage` should return `{ url: '', publicId: '' }` and log a warning (allowing catalog entry creation without image upload in dev)
- Add `CLOUDINARY_CLOUD_NAME`, `CLOUDINARY_API_KEY`, `CLOUDINARY_API_SECRET` to `apps/api/.env.example`

**upload.ts** (`apps/api/src/shared/middleware/upload.ts`):
- Import multer with memoryStorage
- Export `uploadSingle(fieldName: string)` — multer single file, 5MB limit, MIME filter for `image/jpeg`, `image/png`, `image/webp`
- Export `uploadCSV(fieldName: string)` — multer single file, 10MB limit, MIME filter for `text/csv` or `.csv` extension
- Use `multer.memoryStorage()` for both (no disk writes)

**csv.ts** (`apps/api/src/shared/lib/csv.ts`):
- Import Papa from `papaparse`
- Export `parseCSV<T>(buffer: Buffer): { data: T[]; errors: Papa.ParseError[] }` — parses UTF-8 buffer with header: true, skipEmptyLines: true, transformHeader trimming
- Export `generateCSV(rows: Record<string, unknown>[]): string` — wraps Papa.unparse

**slug.ts** (`apps/api/src/shared/utils/slug.ts`):
- Import slugify from `slugify` and `prisma` from shared lib
- Export `slugify(text: string): string` — lower: true, strict: true, locale: 'pt'
- Export `uniqueSlug(text: string, model: 'category' | 'tag' | 'character', excludeId?: string): Promise<string>` — generates slug, checks for collision in the given Prisma model, appends `-N` counter if needed. Uses `prisma[model].findFirst({ where: { slug, ...(excludeId ? { id: { not: excludeId } } : {}) } })` pattern.

Verify: `pnpm --filter api type-check` and `pnpm --filter api build` must pass.
