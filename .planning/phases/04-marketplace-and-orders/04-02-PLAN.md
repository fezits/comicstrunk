<task type="auto">
  <name>Task 1: Create cart service with atomic reservation</name>
  <files>
    apps/api/src/modules/cart/cart.service.ts
  </files>
  <action>
    Create the cart service implementing all CART requirements:

    **addToCart(userId: string, collectionItemId: string):**
    Use `prisma.$transaction` (interactive mode) for atomicity:
    1. Find the collection item (include user); throw NotFoundError if not found
    2. Check `item.isForSale === true`; throw BadRequestError if not for sale
    3. Check `item.userId !== userId`; throw BadRequestError('Cannot add your own item to cart') if same user (CART-06)
    4. Check no existing CartItem with this collectionItemId where expiresAt > now(); throw ConflictError('This item is already reserved') if found (CART-07)
    5. Count user's active cart items (expiresAt > now()); throw BadRequestError('Cart limit reached (maximum 50 items)') if >= 50 (CART-02)
    6. Create CartItem with reservedAt = now(), expiresAt = now + 24h (CART-01)
    7. Return the created cart item with collection item details

    **getCart(userId: string):**
    - Find all CartItems where userId and expiresAt > now()
    - Include collectionItem with catalogEntry (title, coverImageUrl), user (displayName for seller)
    - Order by createdAt desc
    - Return items with calculated remaining time

    **removeFromCart(userId: string, cartItemId: string):**
    - Find CartItem by id; throw NotFoundError if not found
    - Verify cartItem.userId === userId; throw ForbiddenError if mismatch
    - Delete the CartItem
    - Return success

    **clearCart(userId: string):**
    - DeleteMany CartItems where userId and expiresAt > now()
    - Return deleted count

    **getCartSummary(userId: string):**
    - Count active items, sum sale prices from collection items
    - Return { itemCount, totalAmount }
  </action>

  </verify>

  </done>

    **apps/api/src/modules/cart/cart.routes.ts:**
    All routes require authentication (authenticate middleware).

    - `POST /` — Add to cart. Validate body with addToCartSchema from contracts. Call addToCart(req.user.userId, body.collectionItemId). Return 201 with created cart item via sendSuccess.
    - `GET /` — Get cart. Call getCart(req.user.userId). Return cart items array with sendSuccess.
    - `GET /summary` — Get cart summary. Call getCartSummary(req.user.userId). Return { itemCount, totalAmount }.
    - `DELETE /:id` — Remove item from cart. Call removeFromCart(req.user.userId, req.params.id as string). Return sendSuccess with message.
    - `DELETE /` — Clear entire cart. Call clearCart(req.user.userId). Return sendSuccess with deleted count.

    Place static routes (/summary) before parameterized routes (/:id) to avoid Express path collision.

    Export router with `Router` type annotation (TS2742 pattern).

    **apps/api/src/app.ts:**
    - Import cartRouter from modules/cart/cart.routes
    - Mount: `app.use('/api/v1/cart', cartRouter)`
