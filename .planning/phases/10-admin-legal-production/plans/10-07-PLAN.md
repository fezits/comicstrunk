# Plan 10-07: Contact Form UI

## Wave: 2 (depends on 10-03)

## Goal
Public /contact page with category select, submission confirmation, rate limit feedback.

## Tasks

### 1. Create contact API client (`apps/web/src/lib/api/contact.ts`)
- submitContactMessage(data) — public POST

### 2. Create contact components (`apps/web/src/components/features/contact/`)
- `contact-form.tsx` — Form with:
  - Name (Input, required)
  - Email (Input, required, email validation)
  - Category (Select: Sugestão, Problema, Parceria, Outro)
  - Subject (Input, required)
  - Message (Textarea, required, min 10 chars)
  - Submit button "Enviar Mensagem"
  - On success: show confirmation card
  - On rate limit (429): show "Muitas mensagens enviadas. Tente novamente mais tarde."
  - On error: show generic error toast

- `contact-success.tsx` — Confirmation card:
  - Check icon
  - "Mensagem enviada com sucesso!"
  - "Responderemos em até 48 horas úteis."
  - "Voltar ao início" button

### 3. Create route
- `/[locale]/(public)/contact/page.tsx` — Contact form page
  - Title: "Fale Conosco"
  - Contact form component

### 4. Add navigation + footer link
- Add "Contato" to footer links

## Verification
- Form submits successfully
- Confirmation shown after submit
- Rate limit message on 429
- Form validation works (required fields, email format)
