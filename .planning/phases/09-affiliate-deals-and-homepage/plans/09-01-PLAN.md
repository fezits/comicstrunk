# Plan 09-01: Contracts + Partner Store + Deal API

## Wave: 1 (no dependencies)

## Goal
Create Zod contract schemas, PartnerStore CRUD, Deal CRUD with admin workflows, affiliate link composition, and deal expiry cron.

## Tasks

### 1. Create contracts (`packages/contracts/src/deals.ts`)
- DealType enum (COUPON, PROMOTION)
- HomepageSectionType enum (BANNER_CAROUSEL, CATALOG_HIGHLIGHTS, DEALS_OF_DAY, FEATURED_COUPONS)
- createPartnerStoreSchema — name, slug, affiliateTag, baseUrl, logoUrl (optional)
- updatePartnerStoreSchema — all optional
- createDealSchema — storeId, type, title, description, couponCode (optional), discount, affiliateBaseUrl, categoryId (optional), startsAt (optional), expiresAt (optional)
- updateDealSchema — all optional
- listDealsSchema — paginationSchema + filters (storeId, categoryId, type, active, sort)
- listPartnerStoresSchema — paginationSchema + isActive filter
- Export all types + add to index.ts

### 2. Create partner store service (`apps/api/src/modules/deals/partner-stores.service.ts`)
- CRUD: create, getById, update, delete (soft: set isActive=false), list (admin), listActive (public)
- Validate unique slug and name

### 3. Create deal service (`apps/api/src/modules/deals/deals.service.ts`)
- `createDeal(data)` — admin creates deal, validate storeId exists
- `updateDeal(id, data)` — admin updates
- `deleteDeal(id)` — soft delete (isActive=false)
- `getDeal(id)` — single deal with store + category
- `listActiveDeals(filters)` — public: only active + not expired, with store/category filters, sort by newest/discount/expiring-soon
- `listAllDeals(filters)` — admin: all deals including inactive/expired
- `composeDealUrl(deal)` — compose affiliate URL: deal.affiliateBaseUrl + (append affiliateTag from PartnerStore if not already in URL)

### 4. Create routes (`apps/api/src/modules/deals/deals.routes.ts`)
Public routes (no auth):
- `GET /` — listActiveDeals
- `GET /:id` — getDeal

Admin routes (authenticate + authorize ADMIN):
- `GET /admin/list` — listAllDeals
- `POST /admin` — createDeal
- `PUT /admin/:id` — updateDeal
- `DELETE /admin/:id` — deleteDeal
- `POST /admin/:id/banner` — upload banner image (multer + cloudinary)

Partner store routes (admin):
- `GET /stores` — listActive (public)
- `GET /stores/admin/list` — listAll (admin)
- `POST /stores/admin` — create
- `PUT /stores/admin/:id` — update
- `DELETE /stores/admin/:id` — delete

### 5. Register routes in create-app.ts
- Mount at `/api/v1/deals`

### 6. Add deal expiry cron
- In cron/index.ts: daily at 2 AM, deactivate deals where expiresAt < now and isActive = true

### 7. Seed data
- Add to prisma/seed.ts: 3 partner stores (Amazon BR, Mercado Livre, Skoob), 8 deals (mix of COUPON and PROMOTION), some expired
- Rebuild contracts: `pnpm --filter contracts build`

## Verification
- Partner store CRUD works
- Deal CRUD works with image upload
- Public listing filters correctly (active only, by store/category)
- Affiliate URL composition works
- Expired deals hidden from public
