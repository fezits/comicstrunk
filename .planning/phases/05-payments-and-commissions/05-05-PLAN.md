1. Create `apps/web/src/lib/api/banking.ts`:
   - Types:
     ```typescript
     export interface BankAccount {
       id: string;
       userId: string;
       bankName: string;
       branchNumber: string;
       accountNumber: string;
       cpf: string;
       holderName: string;
       accountType: 'CHECKING' | 'SAVINGS';
       isPrimary: boolean;
       createdAt: string;
       updatedAt: string;
     }
     export interface CreateBankAccountInput {
       bankName: string;
       branchNumber: string;
       accountNumber: string;
       cpf: string;
       holderName: string;
       accountType: 'CHECKING' | 'SAVINGS';
       isPrimary?: boolean;
     }
     ```
   - `listBankAccounts(): Promise<BankAccount[]>` — GET `/banking`.
   - `createBankAccount(data: CreateBankAccountInput): Promise<BankAccount>` — POST `/banking`.
   - `updateBankAccount(id: string, data: Partial<CreateBankAccountInput>): Promise<BankAccount>` — PUT `/banking/${id}`.
   - `deleteBankAccount(id: string): Promise<void>` — DELETE `/banking/${id}`.
   - `setPrimaryBankAccount(id: string): Promise<BankAccount>` — PATCH `/banking/${id}/primary`.

2. Create `apps/web/src/components/features/banking/bank-account-form.tsx`:
   - Props: `{ account?: BankAccount; onSubmit: (data: CreateBankAccountInput) => Promise<void>; onCancel: () => void }`.
   - Form fields using shadcn Input, Select, and Label:
     - Banco (bankName) — text input
     - Agencia (branchNumber) — text input
     - Conta (accountNumber) — text input
     - CPF (cpf) — text input with masking: format as `XXX.XXX.XXX-XX` on change using simple onChange handler (same pattern as CEP masking in Phase 4). Only digits stored, display formatted.
     - Titular (holderName) — text input
     - Tipo de conta (accountType) — Select with options "Corrente" (CHECKING) and "Poupanca" (SAVINGS).
     - Conta principal (isPrimary) — Switch/Checkbox toggle.
   - Submit and Cancel buttons.
   - Client-side validation: all fields required except isPrimary, CPF must be 11 digits.
   - Use `useTranslations('banking')`.

3. Create `apps/web/src/components/features/banking/bank-account-list.tsx`:
