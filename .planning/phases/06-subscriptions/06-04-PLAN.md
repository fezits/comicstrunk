  <action>
1. **Add admin schemas to `packages/contracts/src/subscription.ts`**:

   **`adminActivateSubscriptionSchema`**: z.object({
     userId: z.string(),
     planType: z.enum(['FREE', 'BASIC']),
     durationDays: z.number().int().min(1).max(365).optional().default(30),
   })
   Type: `AdminActivateSubscriptionInput`

   **`createPlanConfigSchema`**: z.object({
     planType: z.enum(['FREE', 'BASIC']),
     name: z.string().min(1).max(100),
     price: z.number().min(0),
     billingInterval: z.enum(['MONTHLY', 'QUARTERLY', 'SEMIANNUAL', 'ANNUAL']),
     collectionLimit: z.number().int().min(1),
     commissionRate: z.number().min(0).max(1),
     trialDays: z.number().int().min(0).max(365).optional().default(0),
     isActive: z.boolean().optional().default(true),
     stripePriceId: z.string().optional(),
   })
   Type: `CreatePlanConfigInput`

   **`updatePlanConfigSchema`**: z.object({
     name: z.string().min(1).max(100).optional(),
     price: z.number().min(0).optional(),
     collectionLimit: z.number().int().min(1).optional(),
     commissionRate: z.number().min(0).max(1).optional(),
     trialDays: z.number().int().min(0).max(365).optional(),
     isActive: z.boolean().optional(),
     stripePriceId: z.string().nullable().optional(),
   })
   Type: `UpdatePlanConfigInput`

   **`adminSubscriptionListSchema`**: z.object({
     page: z.coerce.number().int().min(1).optional().default(1),
     limit: z.coerce.number().int().min(1).max(100).optional().default(20),
     status: z.enum(['ACTIVE', 'CANCELLED', 'PAST_DUE', 'TRIALING']).optional(),
     planType: z.enum(['FREE', 'BASIC']).optional(),
   })
   Type: `AdminSubscriptionListInput`

2. **Rebuild contracts**: `pnpm --filter contracts build`

3. **Add admin functions to `apps/api/src/modules/subscriptions/subscriptions.service.ts`**:

   **`adminListSubscriptions(filters: AdminSubscriptionListInput)`**:
   - Build Prisma where clause from filters (status, planType)
   - Query subscriptions with pagination (skip/take), include user (id, name, email, role)
   - Return { data, total, page, limit } (consistent with sendPaginated pattern)

   **`adminActivateSubscription(input: AdminActivateSubscriptionInput)`**:
   - Verify the target user exists
   - Calculate period end: new Date() + durationDays
   - Create a new Subscription record (or update existing active one) using $transaction:
     a. If user has existing ACTIVE/TRIALING subscription, update it
     b. Otherwise create new: userId, planType=input.planType, status='ACTIVE', currentPeriodStart=now, currentPeriodEnd=calculated, stripeCustomerId=null, stripeSubscriptionId=null
     c. Update user role: planType==='BASIC' ? 'SUBSCRIBER' : 'USER' (unless ADMIN)
   - This is the dev mode / admin override path (no Stripe involved)
   - Return the subscription record

   **`adminListAllPlans()`**:
   - Query ALL PlanConfig rows (including inactive), ordered by planType asc, billingInterval asc
   - Return array

   **`adminCreatePlan(data: CreatePlanConfigInput)`**:
   - Create PlanConfig with all fields
   - Return created record

   **`adminUpdatePlan(id: string, data: UpdatePlanConfigInput)`**:
   - Verify plan exists (throw NotFoundError if not)
   - Update PlanConfig with provided fields (spread only defined fields)
   - Return updated record

  <action>
Add admin routes to the existing `subscriptions.routes.ts` file. All admin routes use `authenticate` + `authorize('ADMIN')` middleware.

**Important pattern**: Place static admin routes BEFORE any parameterized routes to prevent Express path collision (consistent with project convention from 02-03 and 04-02 decisions).

Add these routes:

1. **GET `/admin/list`** (authenticate, authorize('ADMIN')):
   - Validate query params with adminSubscriptionListSchema
   - Call adminListSubscriptions(validatedInput)
   - Return via sendPaginated(res, data, { page, limit, total, totalPages })

2. **POST `/admin/activate`** (authenticate, authorize('ADMIN'), validate(adminActivateSubscriptionSchema)):
   - Call adminActivateSubscription(req.body)
   - Return via sendSuccess(res, subscription, 200)

3. **GET `/admin/plans`** (authenticate, authorize('ADMIN')):
   - Call adminListAllPlans()
   - Return via sendSuccess

4. **POST `/admin/plans`** (authenticate, authorize('ADMIN'), validate(createPlanConfigSchema)):
   - Call adminCreatePlan(req.body)
   - Return via sendSuccess(res, plan, 201)

5. **PUT `/admin/plans/:id`** (authenticate, authorize('ADMIN'), validate(updatePlanConfigSchema)):
   - Call adminUpdatePlan(req.params.id as string, req.body)
   - Return via sendSuccess

Import authenticate from shared/middleware/authenticate, authorize from shared/middleware/authorize, validate from shared/middleware/validate.
