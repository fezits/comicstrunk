# Plan 10-06: LGPD + Account Settings UI

## Wave: 2 (depends on 10-02)

## Goal
LGPD rights page, data export download, account deletion flow with consequences.

## Tasks

### 1. Create LGPD API client (`apps/web/src/lib/api/lgpd.ts`)
- createDataRequest(data)
- listMyRequests()
- exportMyData() — returns JSON blob download
- deleteMyAccount()

### 2. Create LGPD components (`apps/web/src/components/features/lgpd/`)
- `lgpd-rights-page.tsx` — Account settings section showing:
  - "Seus Direitos (LGPD)" heading
  - Cards for each right:
    - Acessar dados: "Baixar seus dados" button → triggers export
    - Corrigir dados: "Solicitar correção" button → opens form
    - Exportar dados: "Exportar em JSON" button → same as access
    - Excluir conta: "Excluir minha conta" button → opens deletion flow
  - Request history table: type, status, date

- `data-export-section.tsx` — Export data section:
  - "Exportar Meus Dados" button
  - On click: call exportMyData(), trigger JSON download
  - Show toast "Download iniciado"

- `account-deletion-flow.tsx` — Multi-step deletion:
  - Step 1: Warning card with consequences:
    - "Seus dados pessoais serão removidos em 30 dias"
    - "Pedidos e transações serão mantidos por obrigação fiscal"
    - "Sua coleção e favoritos serão excluídos permanentemente"
    - "Avaliações e comentários serão anonimizados"
  - Step 2: Confirmation: type email to confirm
  - Step 3: Final button "Excluir Minha Conta Permanentemente" (destructive)
  - On success: logout + redirect to homepage

- `correction-request-form.tsx` — Form: details textarea, submit button

### 3. Create routes
- `/[locale]/(collector)/account/lgpd/page.tsx` — LGPD rights page
- Or integrate into existing account settings page

### 4. Add navigation
- Add "Privacidade e Dados" link in account settings navigation

## Verification
- User can export data as JSON download
- User can request account deletion with consequences shown
- Deletion confirmation requires email match
- Request history shows all past requests
